# 种地功能实现计划

## 背景

用户希望为桌面宠物应用添加种地功能，让游戏更加丰富。核心需求包括：

- 右键菜单添加"种地"入口
- 商店可购买种子和肥料
- 种子种植后经历生长阶段
- 用户可浇水、施肥（施肥增加产量）
- 成熟后可收割，未及时收割会枯萎
- 枯萎后仍可收割，但产量降低

## 数据模型设计

### 种子类型 (5种)

| 种子       | 价格 | 生长时间 | 基础产量 | 最大产量 |
| ---------- | ---- | -------- | -------- | -------- |
| 胡萝卜种子 | 10   | 2小时    | 6        | 12       |
| 番茄种子   | 15   | 3小时    | 6        | 15       |
| 玉米种子   | 20   | 4小时    | 9        | 18       |
| 小麦种子   | 12   | 2.5小时  | 12       | 24       |
| 草莓种子   | 30   | 5小时    | 3        | 9        |

### 作物产物

| 作物   | 饱腹恢复 | 出售价格 | 总收益 | 净利润 | 每小时收益 |
| ------ | -------- | -------- | ------ | ------ | ---------- |
| 小麦   | 10       | 4        | 48     | 36     | 14.4       |
| 胡萝卜 | 15       | 14       | 84     | 74     | 37         |
| 番茄   | 20       | 22       | 132    | 117    | 39         |
| 玉米   | 25       | 30       | 270    | 250    | 62.5       |
| 草莓   | 35       | 55       | 165    | 135    | 27         |

**收益设计原则**：

- 每小时收益排序：玉米(62.5) > 番茄(39) > 胡萝卜(37) > 草莓(27) > 小麦(14.4)
- 耗时越长，每小时收益越高
- 小麦产量最高（12个），适合批量种植换取金币总量
- 草莓产量最低（3个）但单件价值最高（饱腹35+售价55），适合需要快速恢复饱腹的场景
- 玉米耗时最长，每小时收益最高，是最高效的金币来源

### 作物状态流转

```
空地 → 种植 → 种子期(0-20%) → 发芽期(20-50%) → 生长期(50-100%) → 成熟期
                                                                      ↓
                                              [收割窗口 = 生长时间 × 20%]
                                                                      ↓
                                                                    枯萎期
```

### 农田规模

- **初始规模**：3x3 (9格)
- **可扩充**：在商店购买扩充
- **最大规模**：9x9 (81格)
- **扩充价格**：
  | 扩充规模 | 价格 | 累计格子数 |
  |---------|------|----------|
  | 3x3 → 4x3 | 100 | 12格 |
  | 4x3 → 4x4 | 200 | 16格 |
  | 4x4 → 5x4 | 400 | 20格 |
  | 5x4 → 5x5 | 600 | 25格 |
  | 5x5 → 6x5 | 800 | 30格 |
  | 6x5 → 6x6 | 1000 | 36格 |
  | 6x6 → 7x6 | 1200 | 42格 |
  | 7x6 → 7x7 | 1500 | 49格 |
  | 7x7 → 8x7 | 1800 | 56格 |
  | 8x7 → 8x8 | 2200 | 64格 |
  | 8x8 → 9x8 | 2600 | 72格 |
  | 9x8 → 9x9 | 3000 | 81格 |

### 游戏机制

- **水分系统**：初始100，每分钟衰减1，低于30生长暂停
- **浇水**：每次+30水分，上限100，冷却10秒
- **施肥**：每次+10%产量，最多3次，仅种子期/发芽期有效，肥料价格15金币/个
- **收割窗口**：根据作物生长时间动态计算，窗口时长 = 生长时间 × 20%（如5小时作物有1小时收割窗口）
- **枯萎惩罚**：超过收割窗口未收割则枯萎，产量-50%
- **收割产出**：作物 + 有概率获得额外种子（概率30%）

### 天气系统联动

| 天气      | 水分变化 | 生长速度 | 收割产量 | 备注             |
| --------- | -------- | -------- | -------- | ---------------- |
| 默认/晴天 | 每分钟-1 | 正常     | 正常     | 基准状态         |
| 多云      | 每分钟-1 | 正常     | 正常     | 正常状态         |
| 小雨      | 不衰减   | 正常     | 正常     | 自然浇水         |
| 暴雨      | 每分钟+5 | 正常     | 正常     | 水分自动增加     |
| 雷阵雨    | 不衰减   | 正常     | 正常     | 自然浇水         |
| 小雪      | 不衰减   | 减慢20%  | -30%     | 冬季作物生长缓慢 |
| 大雪      | 不衰减   | 减慢30%  | -30%     | 严寒影响生长     |

- **天气检测**：每10分钟检测一次当前天气，更新农田状态
